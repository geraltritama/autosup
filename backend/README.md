# AUTOSUP Backend API 🚀

Backend system untuk platform Supply Chain berbasis AI (Hackathon Colosseum Frontier).

## 🛠️ Tech Stack

- **Framework:** FastAPI
- **Database:** Supabase
- **AI:** Google Gemini AI
- **Deployment:** Railway

## 📦 Quick Start (Local)

1. Install dependencies: `pip install -r requirements.txt`
2. Jalankan server: `uvicorn main:app --reload`
3. Dokumentasi API: `http://127.0.0.1:8000/docs`

## 🚂 Deployment

Project ini siap di-deploy ke Railway dengan menghubungkan repository GitHub. Pastikan Environment Variables (.env) sudah diatur di panel Railway.

## 🤝 Note untuk Frontend (Geral)

- API Base URL akan diberikan setelah deployment.
- Tidak perlu merge folder, cukup panggil API menggunakan link publik.
